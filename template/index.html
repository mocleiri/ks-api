Page Template

  • template.html - Base layout structure. Includes header, body, and footer.

CSS

  • reset.css - Eric Meyer's Reset CSS. Resets all CSS for browser consistency.
  • fonts.css - Base styling for fonts. Determines default font sizes and colors.
  • main.css - Contains most of the general CSS used for layout and miscellaneous.
  • tabnavigation.css - CSS specific to the tab navigation and sub navigation.
  • forms.css - Snippet: CSS specific to forms and form elements.
  • buttons.css - CSS specific to Button elements.
  • tables.css - CSS specific to Table elements.
